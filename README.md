# StethoTest
[Stetho](http://facebook.github.io/stetho/) is a sophisticated debug bridge for Android applications.  

Here you can find a blog about this eccellent tool:
http://gmariotti.blogspot.com/2015/07/a-first-glance-at-stetho-tool.html

This repo is only used to test the tool as described in the blog.
